## 1. 회사 기본 정보

- **회사명**: 파라메타 (PARAMETA)  
- **한 줄 소개**:  
  전통 금융(TradFi)과 디지털 자산(DeFi)을 연결하는 규제 친화적 Web3 금융 인프라 기업  

- **설립 배경**:  
  기존 금융과 디지털 자산이 제도·책임 구조 문제로 분리되어 있었으며, 이를 연결하는 현실적인 인프라 설계 필요성에서 출발  

- **핵심 비전**:  
  Wallet 중심 금융 구조(WalletFi)로 전환되는 시대에서 금융 인프라 레이어 선점

## 2. 주요 제품 / 서비스

### 2.1 ParaSTA (핵심 플랫폼)

#### 개요
ParaSTA는 스테이블코인 및 디지털 자산 환경에서  
발행–지갑–정산–KYC–운영까지 통합 제공하는 금융 인프라 플랫폼

#### 핵심 구조 (5대 모듈)

1. **Issuance (발행)**
   - 스테이블코인 발행/소각
   - 준비금 관리 및 PoR
   - 스마트컨트랙트 기반 라이프사이클 통제

2. **Wallet (지갑)**
   - MPC/KMS 기반 키 관리
   - 계정추상화 기반 UX
   - 멀티체인 자산 통합 관리

3. **Orchestration (연동)**
   - 은행 계좌 ↔ 블록체인 자산 통합
   - 자동 환전 및 정산
   - API 기반 금융 자동화

4. **On-chain KYC**
   - DID/VC 기반 신원 인증
   - AML/트래블룰 자동 적용
   - 멀티체인 인증

5. **Unified Admin**
   - 통합 관제 및 운영 관리
   - 감사 로그 및 리포팅
   - 실시간 리스크 관리

#### 주요 가치

- 규제 대응 내재화 (AML / KYC / Audit)
- 금융망과 블록체인 통합
- API 기반 자동화
- 단계별 도입 구조 (PoC → Pilot → Commercial)

### Solution
